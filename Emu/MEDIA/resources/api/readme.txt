GVU API Keys
============

spruceOS does NOT include any API keys with GVU. To enable cover art
scraping and subtitle search, you need to get your own free keys and
add them to this directory.

SubDL_API.txt
  Your SubDL API key for subtitle search.
  Get one free at: https://subdl.com/
  Format: paste the key on a single line, no extra text.

TMDB_API.txt
  Your TMDB API key (v3 read access token) for cover art.
  Get one free at: https://www.themoviedb.org/settings/api
  Format: paste the key on a single line, no extra text.

Without these keys:
- Cover art scraping uses TVMaze only (no key needed, fewer results).
- Subtitle search will not work.
